## Naming Rules for Configuration Directories / 配置目录命名规则

The naming convention for configuration directories follows the `sourceRepositoryName_branchName` format. For example, `openwrt_openwrt-24.10` indicates that the build uses the `openwrt-24.10` branch from the https://github.com/openwrt/openwrt repository.

配置目录采用 OpenWrt 在 github.com 上的 `源码仓库名_分支名` 作为命名规则。例如，`openwrt_openwrt-24.10` 表示使用 https://github.com/openwrt/openwrt 仓库的 `openwrt-24.10` 分支进行编译。
